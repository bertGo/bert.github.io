- webkit-box-orient 指定元素水平排列

  ```css
  //指定文本在第二行截断以省略号结束的规范代码
  p{
                       display: -webkit-box;
                      -webkit-box-orient: vertical;
                       -webkit-line-clamp: 2;           
                       text-overflow: ellipsis;
                       overflow: hidden;
                          // height:50px ;
                      // width:216px;本例不需要
  }
  ```
